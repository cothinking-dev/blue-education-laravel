# Blue Education Website Content Extraction

**Extraction Date:** January 8, 2026
**Source:** https://www.blueeducation.com.au/

## Overview

This directory contains extracted content from all main pages of the Blue Education website. The content has been converted to markdown format with clean formatting, appropriate headings, lists, and paragraphs. Navigation menus, headers, footers, and internal/external links have been removed.

## Extraction Methodology

Due to the website being built on the Wix platform with heavy JavaScript rendering, direct web scraping of the HTML was not feasible. Instead, content was extracted through:
- Web search indexing of page content
- Cached search results
- Multiple search queries targeting specific pages and topics
